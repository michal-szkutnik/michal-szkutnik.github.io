Canon RAW files (CR2 extension) missing thumbnails on Windows 10 (probably applies to Windows 8 also) after upgrade from Windows 7 with SageThumbs installed

I had SageThumbs installed on Windows 7 to get CR2 file thumbnails. After upgrade to Windows 10 the thumbnails failed to generate for most of the files. Ultimately I uninstalled SageThumbs and was left with no CR2 thumbnails. After some digging it turned out I was missing a ShellEx registry entry pointing to Windows built in thumbnail generator for CR2 files. If your CR2 files are missing thumbnails in File Explorer and no tips found on the internet seem to fix it, maybe it's the same issue. Check if your HKEY_CLASSES_ROOT.cr2 key looks like that (especially the HKEY_CLASSES_ROOT.cr2\ShellEx{e357fccd-a995-4576-b01f-234630154e96} key):

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.cr2]
@="Canon RAW File"
"PerceivedType"="image"
"Content Type"="image/cr2"

[HKEY_CLASSES_ROOT\.cr2\ShellEx]

[HKEY_CLASSES_ROOT\.cr2\ShellEx\{e357fccd-a995-4576-b01f-234630154e96}]
@="{C7657C4A-9F68-40fa-A4DF-96BC08EB3551}"

If not, adding these values and restarting the machine might fix your problem.
